=====================
AneCMS Mobile
=====================

An Android Application that interacts with AneCMS through XML-RCP

Features
========

* Display Site Name and Available webservices
* Blog Module, read and post articles

Requirements
============

* Android >= 2.1
* Older Platforms maybe...
